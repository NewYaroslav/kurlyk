Kurlyk Library

Introduction

Kurlyk is a library designed for making HTTP requests easy and efficient. It abstracts the complexity of handling asynchronous HTTP operations and provides a simple interface for developers.

Features

  • Asynchronous and synchronous HTTP requests.
  • Rate limiting to control the number of requests over time.
  • Simple API for GET and POST requests.
  • Easy integration with existing C++ projects.

Usage

#include <kurlyk.hpp>
#include <iostream>
int main() {
kurlyk::init(true);
// Asynchronous GET request using the standalone function
uint64_t request_id = kurlyk::http_get("https://httpbin.org/ip", kurlyk::QueryParams(), kurlyk::Headers(),
[](const kurlyk::HttpResponsePtr response) {
print_response(response);
});
std::system("pause");
kurlyk::cancel_request_by_id(request_id).wait();
kurlyk::deinit();
return 0;
}

Installation

To use the Kurlyk Library in your project, include the header file kurlyk/kurlyk.hpp in your source code. Ensure that your project is set up to compile with standards from C++11 to C++17.
